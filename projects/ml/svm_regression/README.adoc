== SVM Regression Example
This example demonstrates Support Vector Machine (SVM) regression using a linear kernel.
The dataset consists of two-dimensional points labeled with binary targets (0 or 1). The SVM model is trained to classify these points based on their features.

=== Example1: Classifying linearly separable data using SVM
Learning and inference are performed using a linear SVM model.

[source,shell]
.Run the SVM classification example
----
cargo run --bin svm_linear_classify --release
----
