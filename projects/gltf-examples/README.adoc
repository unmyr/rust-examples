== GLTF Examples

This directory contains a collection of example glTF files demonstrating various features of the glTF format. For more information about the glTF format and its specifications, you can refer to the following resources:

* https://docs.rs/gltf-json/latest/gltf_json/[Crate gltf_json^]
* https://github.com/gltf-rs/gltf/blob/main/examples/export/main.rs[gltf-rs Export Example^]

== Run examples

1. Generate glTF files
+
[source,shell]
----
cargo run --bin gen_gltf_triangle_grayscale --release
cargo run --bin gen_gltf_triangle_v_colors --release
cargo run --bin gen_gltf_triangle_red --release
----

2. View glTF files
+
[source,shell]
----
cargo run --bin gltf_viewer --release
----
