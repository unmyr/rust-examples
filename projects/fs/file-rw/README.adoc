=== Set up a new project
. Set up a new project, and go to the projects directory.
+
[source,console]
----
$ cargo new --bin file-rw
     Created binary (application) `file-rw` package
$ cd file-rw/
----

=== Run tests

[source,console]
.Results
----
$ cargo test
...
----
