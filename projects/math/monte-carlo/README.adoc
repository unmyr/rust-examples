=== Create a new project

. Let's create a new library project called `monte-carlo`
+
[source,console]
----
$ cargo new --lib monte-carlo
     Created library `monte-carlo` package
$ cd monte-carlo/
----

=== Tests

. The cargo test command runs all tests in our project.
+
[source,shell]
----
cargo test
----
