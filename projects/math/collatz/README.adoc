=== Create a new project

. Let's create a new library project called `collatz`
+
[source,console]
----
$ cargo new collatz --lib
     Created library `collatz` package
$ cd collatz/
----

=== Tests

. The cargo test command runs all tests in our project.
+
[source,console]
----
$ cargo test
...
----
