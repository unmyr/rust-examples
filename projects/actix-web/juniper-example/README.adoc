=== Set up a new project
. Set up a new project, and go to the projects directory.
+
[source,console]
----
$ cargo new --bin juniper-example
     Created binary (application) `juniper-example` package
$ cd juniper-example/
----

[source,console]
----
$ cargo run

     Running `target/debug/simple-server`
----

----
{
  human(id: "1234") {
    name
    appearsIn
    homePlanet
  }
}
----
