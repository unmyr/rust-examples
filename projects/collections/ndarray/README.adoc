== ndarray examples

=== Running the examples
To run the examples in this directory, use the following command:

[source,bash]
----
cargo run --bin matrix_multiplication --release
----
This will execute the `matrix_multiplication` example in release mode.

[source,plaintext]
.Command execution results
----
[[2, 3],
 [1, 2]]
2 3
1 2
----
