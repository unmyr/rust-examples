=== Updating a String

==== Appending to a String with push_str and push
[source,rust]
.Using a string slice after appending its contents to a `String`
----
fn main() {
    let mut s1 = String::from("foo");
    let s2 = "bar";
    s1.push_str(s2);
    println!("s2 is {}", s2);
}
----

[source,console]
----
